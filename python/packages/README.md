# AutoGen Python Packages 快速索引

以下列出 `python/packages/` 內常用套件及其中文導覽，協助在處理程式碼或技術問題時快速定位對應功能說明。詳盡操作與範例請參考各套件的 `README-zh.md` 與官方文件。

- [agbench](agbench/README-zh.md)：基準測試工具，提供 Docker/原生重播與結果匯整流程。
- [autogen-core](autogen-core/README-zh.md)：事件驅動的多代理核心 Runtime 與組件系統。
- [autogen-agentchat](autogen-agentchat/README-zh.md)：高階多代理 API、訊息模型與團隊編排。
- [autogen-ext](autogen-ext/README-zh.md)：官方維護的擴充代理、模型客戶端與程式執行器。
- [autogen-magentic-one](autogen-magentic-one/README-zh.md)：舊版 Magentic-One 參考實作與遷移指引。
- [autogen-studio](autogen-studio/README-zh.md)：AutoGen Studio 後端服務、Lite 模式與資料庫管理。
- [autogen-test-utils](autogen-test-utils/README-zh.md)：OpenTelemetry 測試工具與遙測檢驗流程。
- [component-schema-gen](component-schema-gen/README-zh.md)：產生 AutoGen Component JSON Schema 的 CLI。
- [magentic-one-cli](magentic-one-cli/README-zh.md)：命令列介面，快速啟動 Magentic-One 團隊。
- [pyautogen](pyautogen/README-zh.md)：AutoGen 代理套件代理入口與版本遷移資訊。

更多 AutoGen 官方教學、API 與設計文件請參考：[https://microsoft.github.io/autogen/stable/index.html](https://microsoft.github.io/autogen/stable/index.html)
